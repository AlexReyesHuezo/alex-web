# Getting Started with Astro

Astro is an all-in-one web framework for building fast, content-focused websites.

## Why Astro?

1. **Performance**: Astro ships zero JavaScript to the client by default.
2. **Islands Architecture**: Load interactive components only when needed.
3. **Content Collections**: Type-safe markdown content.
